Seatwork Activity

Here are some of the activities that I have done in my web design with client side scripting .

Seatwork 1

Understanding Website

This activity aims to familiarize students with website components by dissecting a chosen design. Students will create a wireframe based on their selected website design to visualize its structure and layout effectively.

View

Seatwork 2

Resume website

Requires students to select a complete one-page website or landing page design from platforms like Pinterest or Behance. The chosen design must include a header, footer, and various sections.

View

Seatwork 3

Headers and Footers Display Website

Create a Header and Footer Display Website with distinct designs using pure HTML and CSS. Include both static and animated versions of the headers and footers.

View

Seatwork 4

Hero Display Website

This project involves creating a hero display website using pure HTML and CSS. The website will showcase distinct design layouts for various features, emphasizing different layout techniques. Multiple pages will be included to demonstrate various design styles and principles.
